# l10n

## Valid

### enables localization after load

```htmldjango
{% load l10n %}

{% localize on %}
  {{ value }}
{% endlocalize %}
```

```snapshot
✓ no diagnostics
```

### disables localization after load

```htmldjango
{% load l10n %}

{% localize off %}
  {{ value }}
{% endlocalize %}
```

```snapshot
✓ no diagnostics
```

## Invalid

### localize requires load when l10n is not configured as a builtin

```htmldjango
{% localize on %}
  {{ value }}
{% endlocalize %}
```

```snapshot
error[S109]: Tag 'localize' requires {% load l10n %}
 --> test.html:1:1
  |
1 | {% localize on %}
  | ^^^^^^^^^^^^^^^^^
```

### reports unclosed localize block

```htmldjango
{% load l10n %}
{% localize on %}
  <p>Never closed.</p>
```

```snapshot
error[S100]: Unclosed tag: localize
 --> test.html:2:1
  |
2 | {% localize on %}
  | ^^^^^^^^^^^^^^^^^
```
